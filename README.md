# blueJhrLibrary
Para usar la libreria BlueJhr utilizada para microcontroladores sigue los siguientes pasos

1) Importar la libreria BlueJhr 
 ```
 allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
 ```
